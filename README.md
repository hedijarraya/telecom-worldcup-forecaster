# World Cup Telecom Demand Forecaster & Insight Engine

### Goal
Predict regional telecom traffic surges during FIFA World Cup 2026 matches and build an automated capacity dashboard that explains network spikes using AI-generated narratives.

### Inputs
* **Match Schedule & Context:** Kickoff times, competing countries, and match importance (e.g., Group Stage vs. Finals).
* **Demographics & Baseline Data:** Country population, internet penetration rates, and historical baseline network traffic.

### Expected Outputs
* **Demand Forecasts:** Predicted time-series traffic spikes (streaming and data consumption) during live matches.
* **Capacity Dashboard:** Visualizations mapping regional network loads and predicted peak demand.
* **Auto-Narratives:** Automated, LLM-generated summaries explaining the specific context behind each traffic surge.
